# ColorWars PWA Prototype

Лёгкий браузерный прототип игры в духе ColorWars. Игрок выбирает цвет, расширяет свою территорию по соседним клеткам и соревнуется с тремя ботами за большую часть поля.

## Почему PWA подходит для MVP

Для первого прототипа достаточно статической PWA-архитектуры:

- нет сервера и сборщика, игру можно открыть через любой статический хостинг;
- `index.html` содержит UI, правила, игровое состояние и рендер;
- `manifest.webmanifest` делает приложение устанавливаемым;
- `sw.js` кэширует app shell для офлайн-запуска;
- прогресс матча и статистика сохраняются в `localStorage`.

Такой подход хорошо подходит для проверки core loop: выбор цвета -> захват территории -> ход AI -> сравнение результата.

## Как запустить

Нужен любой локальный HTTP-сервер, потому что service worker не работает с прямым открытием файла.

```bash
python3 -m http.server 4173
```

Затем откройте:

```text
http://localhost:4173
```

## Текущая механика

- поле 13x13;
- 4 игрока: человек и 3 AI;
- игрок выбирает цвет из палитры;
- вся его территория меняет цвет;
- соседние клетки того же цвета присоединяются к территории;
- боты после хода игрока выбирают цвет с наибольшей ожидаемой выгодой;
- матч завершается, когда поле заполнено или закончились 32 раунда;
- статистика побед, игр и лучшего процента территории сохраняется локально.
